# RevisitingSSI.com

Website for the #RevisitingSSI workshops - exploring harms and opportunities overlooked by mainstream Self-Sovereign Identity.

## About

This site hosts information about the #RevisitingSSI workshop series, which focuses on:

- **Anti-Coercion**: Preventing manipulation and preserving cognitive liberty
- **Voluntary Constraint**: Understanding productive vs. exploitative participation
- **Relational Autonomy**: Supporting relationships and community credentials in identity systems

## Technical Details

This site is built using:
- [Jekyll](https://jekyllrb.com/) static site generator
- [BaseTheme](https://github.com/ChristopherA/BaseTheme) remote theme
- GitHub Pages for hosting

## Repository Structure

This repository uses git worktree architecture:
- Bare repository: `RevisitingSSI.com.git/`
- Main branch worktree: `main/`

## Local Development

To run locally (optional - GitHub Pages builds automatically):

```bash
bundle install
jekyll serve
```

Visit http://localhost:4000
