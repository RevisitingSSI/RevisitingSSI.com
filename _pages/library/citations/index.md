# Citations Library

This library consolidates all citations referenced across the Revisiting Self-Sovereign Identity lens briefs. Browse citations in different ways:

## Views

### [By Lens](/library/citations/by-lens/)

Browse citations organized by the 15 lens briefs. Each lens provides context, descriptions, and links to its supporting citations. Best for exploring how sources inform specific perspectives.

### [All Citations (Alphabetical)](/library/citations/all/)

Complete alphabetical reference with full bibliographic information, tags, abstracts, and aggregated "Why This Matters" sections showing cross-lens relevance. Best for looking up specific sources or understanding how a single work informs multiple lenses.

### [By Year](/library/citations/by-year/)

Browse citations organized chronologically, most recent first. See how the field has evolved from foundational works (1948-1990s) through contemporary contributions (2020s).

## About This Library

Each citation includes:
- **Full bibliographic information** with retrieval URLs
- **Tags** for topic categorization
- **Brief** one-line description
- **Short Abstract** summarizing the work
- **Why This Matters** explaining relevance to each lens that references it

Citations appear in multiple lens contexts but are defined once in the [All Citations](/library/citations/all/) reference, ensuring consistency while showing cross-lens connections.
